栈帧：一个方法一个栈帧

方法的结束方式分为两种：

  1. 正常结束
  2. 出现未捕获的异常

栈帧的内部结构

每个栈帧储存着：

  1. 局部变量表
  2. 操作数栈
  3. 动态链接
  4. 方法返回地址
  5. 一些附加信息

局部变量表（local variables）

  1. 数字数组，用于储存局部变量的
  2. 无数据安全问题，是建立在线程的栈上，是线程的私有数据
  3. 容量在编译期确定下来，就不会更改

Solt 局部变量表最基本的储存单元

非静态变量：this 放在solt = 0 的位置上
