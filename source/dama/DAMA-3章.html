二、多选

1、度量数据价值的方式包含（）答案：ABCDE

A.替换成本（Replacement Cost）。在灾难性数据破坏事件或者数据中断时，数据替换或恢复的成本，包括组织内的交易、域、目录、文档和指标信息等。

B.市场价值（Market Value）。兼并或收购企业时作为企业资产的价值。

C.发现商机（Identified Opportunities）。通过交易数据或者通过售卖数据，从数据（商务智能）中发现商机获得的收入价值。

D.售卖数据（Selling Data）。一些组织为产品或销售将数据打包从数据中获得的洞察。

E.风险成本（Risk Cost）。它是基于潜在罚款、补救成本和诉讼费用的估价。

3、数据管理岗位的类型,以下说法正确的有（）答案：ABCDEFG

A.首席数据管理专员（Chief Data Stewards）。CDO的替代角色，担任数据治理机构的主席，也可以是虚拟的（基于委员会）或者在分布式数据治理组织中担任CDO。他们甚至也可能是高层发起者。

B.高级数据管理专员（Executive Data Stewards）。他们是数据治理委员会（DGC）的资深管理者。

C.企业数据管理专员（Enterprise Data Stewards）。他们负责监督跨越业务领域的数据职能。

D.业务数据管理专员（Business Data Stewards）。他们是业务领域专业人士，通常是公认的领域专家，对一个数据域负责。他们和利益相关方共同定义和控制数据。

E.数据所有者（Data Owner）。他们是某个业务数据管理专员， 对其领域内的数据有决策权。

F.技术数据管理专员（Technical Data Stewards）。他们是某个知识领域内工作的IT专业人员，如数据集成专家、数据库管理员、商务智能专家、数据质量分析师或元数据管理员。

G.协调数据管理专员（Coordinating Data Stewards）。这在大型组织中尤为重要，其领导并代表业务数据管理专员和技术数据管理专员进行跨团队或者数据专员之间的讨论。

7、数据治理通过建立业务术语表，用于定义和定位组织中的数据；确保组织中数量繁多的元数据被管理和应用。 答案：正确

9、数据治理确保数据被恰当地管理而不是直接管理数据。答案：正确
